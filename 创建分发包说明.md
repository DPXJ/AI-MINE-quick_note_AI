# QuickNote AI - 创建分发压缩包说明

## 📦 快速创建分发包

### 方法一：使用自动脚本（推荐）

1. **运行打包脚本**
   ```bash
   一键打包.bat
   ```
   这个脚本会自动：
   - 打包程序为exe
   - 创建zip压缩包
   - 版本号自动递增

2. **运行分发包创建脚本**
   ```bash
   创建分发包.bat
   ```
   这个脚本会：
   - 查找最新打包的版本
   - 创建包含所有必要文件的分发包
   - 生成zip压缩包

### 方法二：手动创建分发包

如果自动脚本无法运行，可以手动创建：

#### 步骤1：打包程序
```bash
一键打包.bat
```
或
```bash
pyinstaller build.spec
```

#### 步骤2：创建分发包目录
1. 在 `dist` 目录下创建新文件夹：`分发包_QuickNote_AI_vX.XX`
2. 将打包好的程序文件夹（如 `QuickNote_AI_v0.32`）复制到分发包目录中

#### 步骤3：添加配置文件模板
将以下文件复制到分发包根目录：
- `config.yaml` → 重命名为 `config.yaml.example`
- `env.example` → 重命名为 `.env.example`

#### 步骤4：创建使用说明
创建 `README.md` 文件，包含：
- 快速开始指南
- 配置文件说明
- 常见问题解答

#### 步骤5：创建压缩包
1. 选中整个分发包文件夹
2. 右键 → 发送到 → 压缩(zipped)文件夹
3. 重命名为 `QuickNote_AI_分发包_vX.XX.zip`

## 📋 分发包应包含的文件

```
QuickNote_AI_分发包_vX.XX/
├── QuickNote_AI_vX.XX/          # 主程序文件夹
│   ├── QuickNote_AI_vX.XX.exe   # 主程序
│   ├── _internal/               # 依赖文件（必需）
│   └── logs/                    # 日志目录（自动创建）
├── config.yaml.example          # 配置文件模板
├── .env.example                 # 环境变量配置模板
└── README.md                    # 使用说明
```

## ✅ 检查清单

创建分发包后，请确认：

- [ ] 包含完整的程序文件夹（含 `_internal` 目录）
- [ ] 包含 `config.yaml.example` 配置文件模板
- [ ] 包含 `.env.example` 环境变量模板
- [ ] 包含 `README.md` 使用说明
- [ ] 压缩包可以正常解压
- [ ] 解压后的程序可以正常运行（需要先配置 `.env`）

## 🚀 分发说明

将压缩包发送给其他用户时，提醒他们：

1. **解压文件**到任意位置（建议英文路径）
2. **配置API密钥**：
   - 复制 `.env.example` 为 `.env` 并填入密钥
   - 复制 `config.yaml.example` 为 `config.yaml`
3. **运行程序**：
   - 进入 `QuickNote_AI_vX.XX` 文件夹
   - 双击 `QuickNote_AI_vX.XX.exe`
4. **以管理员身份运行**（全局快捷键需要）

## 📝 注意事项

- 确保路径中没有中文字符
- 首次运行需要配置 `.env` 文件
- 全局快捷键功能需要管理员权限
- 可能需要安装 Visual C++ Redistributable
