## 🔧 如果遇到模块导入错误

如果仍然遇到 `ModuleNotFoundError: No module named 'src'`，请：

1. **确认在项目根目录运行**
   ```bash
   cd "d:\Lan_Company\03-AI-MINE\02-同步flomo"
   ```

2. **检查当前目录**
   ```bash
   # 在命令行执行
   dir
   # 应该能看到 src 目录
   ```

3. **手动设置PYTHONPATH**
   ```bash
   set PYTHONPATH=%CD%
   python -c "from src.main import main; main()"
   ```
